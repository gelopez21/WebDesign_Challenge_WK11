Summary: Latitude vs. X

The purpose of this project was to analyze how changes as you get closer to the equator.
To accomplish this analysis, I pulled the data from Weather Map API to assemble a dataset on over 500 cities.
After assembling the data, I used Matplotlib to look at temps, cloudiness, wind speed, and humidity.
